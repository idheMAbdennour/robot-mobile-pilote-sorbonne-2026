# robot-mobile-pilote-sorbonne-2026
Projet 2026 de L3 Electronique de Sorbonne Université : Réalisation d'un système complexe de livraison de colis par robot dans une usine.

Le projet est divisé en 4 sous projets :
Robot, Poste, Centrale, et Supervision. Chaque sous-projet possède un dossier de fichiers sources de développement unitaire de chaque sous-modules, et un dossier de test qui contiendra les sources pour les unités de tests indépendantes associées à chaque sous-modules. Un projet Keil se trouve à la racine de chaque sous-projet.
