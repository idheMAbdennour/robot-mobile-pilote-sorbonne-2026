// Ce fichier décrit les elements interactifs de l'interface de l'application de supervision

L'opérateur peut intéragir avec trois elements de l'interface graphique :
    - Les cases au contour bleu contenant un point d'interrogation dans la colonne "Robot" du sous-écran 2 "GESTIONNAIRE DE MISSION" : Case attribution. (Les cases de cette colonne contenant un numero NE SONT PAS CLIQUABLE)
    - Les cases contenant une liste déroulante de la colonne "Status" du sous-écran 2 "GESTIONNAIRE DE MISSION" : Case vitesse. (Les cases de cette colonne contenant un point d'interrogation NE SONT PAS CLIQUABLE)
    - Les cases de la matrice de robots du sous-écran 3 "LISTE ROBOTS" : Case robot.

Ces cases sont des élements clickable.
